# 统一画板尺寸（新增）

# 间距 (新增)

- 为各种字号指定了搭配的行高，方便字体与其他元素做对齐

- 控件级别的尺寸

- 控件和 UI 元素之间的间距

- 组件之间的间距

- 网格栏间距，页边距，区域间距等页面布局间距

> 参考 [Clarity Design - 间距](https://design.teambition.com/visual-spacing)

> 参考 [Material Design - 间距](https://material.io/design/layout/spacing-methods.html#)
